# 第 5 章 系统测试与分析

## 5.1 引言
*   简述本章目的：验证所提算法在准确性、鲁棒性、实时性及细粒度分类能力上的综合表现。
*   概括实验内容：数据集介绍、评价指标、SOTA对比、消融实验、鲁棒性分析及工程部署测试。

## 5.2 实验环境与配置
### 5.2.1 数据集介绍与预处理
*   **CULane 数据集**：重点介绍其 9 大复杂场景（拥堵、夜间、无标线等），作为鲁棒性测试基准。
*   **OpenLane 数据集**：重点介绍其 14 种车道线类别和拓扑关系，作为细粒度分类和 3D 感知的测试基准。
*   **TuSimple 数据集**（可选）：作为简单场景的基准参考。
*   **数据增强策略**：介绍训练时使用的 Mosaic、Mixup 或几何变换策略。

### 5.2.2 评价指标体系
*   **检测指标**：F1-Score, Precision, Recall, TP/FP/FN 定义（基于 IoU 阈值）。
*   **分类指标**：Category Accuracy (类别准确率), Attribute Accuracy (属性准确率)。
*   **几何指标**：x/y 误差，角度误差 $\theta_{err}$。
*   **效率指标**：FPS (帧率), Latency (延迟), FLOPs, Params, Model Size (MB)。

### 5.2.3 训练细节与平台
*   **硬件环境**：训练端 (NVIDIA RTX 4090/3090)，推理端 (NVIDIA Jetson AGX Orin)。
*   **超参数设置**：Optimizer (AdamW), Learning Rate Schedule (Cosine Annealing), Batch Size, Epochs, Loss Weights ($w_{cls}, w_{reg}, w_{iou}$ 等)。
